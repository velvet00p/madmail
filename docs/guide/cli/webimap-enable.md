# `madmail webimap enable`

Parent: [`webimap`](webimap.md)

Enable the API

## Synopsis

```bash
madmail webimap enable [OPTIONS]
```


After changes, run `madmail reload` (or restart) to apply.

## JSON output (`--json`)

```bash
madmail webimap enable --json
```

Success stdout:

```json
{"ok": true, "command": "webimap enable", "data": { ... }}
```
