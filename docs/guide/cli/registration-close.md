# `madmail registration close`

Parent: [`registration`](registration.md)

Block new registrations

## Synopsis

```bash
madmail registration close [OPTIONS]
```

## JSON output (`--json`)

```bash
madmail registration close --json
```

Success stdout:

```json
{"ok": true, "command": "registration close", "data": { ... }}
```
