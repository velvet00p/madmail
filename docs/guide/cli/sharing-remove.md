# `madmail sharing remove`

Parent: [`sharing`](sharing.md)

Remove a share link

## Synopsis

```bash
madmail sharing remove [OPTIONS] <SLUG>
```


## Notes

Alias: `madmail sharing delete <SLUG>`.

## JSON output (`--json`)

```bash
madmail sharing remove --json
```

Success stdout:

```json
{"ok": true, "command": "sharing remove", "data": { ... }}
```
